# Student-Marks-Analyzer-NumPy


## Features
- Takes marks of students as input
- Stores data using NumPy arrays
- Displays:
  - Shape
  - Size
  - Dimensions
  - Data Type
- Calculates:
  - Total Marks
  - Average Marks
  - Highest Marks
  - Lowest Marks
- Counts:
  - Passed Students
  - Failed Students
  - A Grade Students
- Converts marks to float
- Assigns grades to each student

## Technologies Used
- Python
- NumPy
